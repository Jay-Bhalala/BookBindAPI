# BookBind API

## Overview
BookBind API is a RESTful service designed to support bookstore applications. It features over 30 endpoints that allow for the creation of new orders, updating user profiles, modifying book reviews, and more. This project utilizes Python with Flask for the server-side logic, MongoDB for the database, and JWT for secure authentication.

## Features
- User Authentication using JWT
- CRUD operations for books, orders, and user profiles
- Comprehensive API documentation using Mintlify

## Technologies
- Python 3.9+
- Flask
- MongoDB
- JWT (JSON Web Tokens)
- Node.js (for Mintlify documentation)
- Mintlify for API documentation

## Setup

### Prerequisites
- Python 3.9 or higher
- MongoDB running on the default port
- Node.js (for Mintlify)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/BookBindAPI.git
   cd BookBindAPI

2. Create and activate a virtual environment & install the requirements:
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows use `venv\Scripts\activate`
   pip install -r requirements.txt

3. Create a .env file in the root directory and populate it with the necessary environment variables and run the application:
   ```bash
   MONGO_URI=mongodb://localhost:27017/bookbind
   SECRET_KEY=your_secret_key
   python run.py

3. To test the API, run the automated tests for this system:
   ```bash
   python -m unittest discover -s tests

## Documentation
Access the API documentation generated by Mintlify at `http://localhost:5000/docs` after running the application.

## BookBind API Endpoints

### User Management

- **Register a New User**
  - `POST /users/register`
  - Registers a new user account.

- **User Login**
  - `POST /users/login`
  - Authenticates a user and returns a JWT.

- **User Logout**
  - `POST /users/logout`
  - Logs out a user, invalidating the JWT.

- **Get User Profile**
  - `GET /users/profile`
  - Retrieves the profile of the authenticated user.

- **Update User Profile**
  - `PUT /users/profile`
  - Updates the profile information of the authenticated user.

- **Delete User Account**
  - `DELETE /users/profile`
  - Deletes the authenticated user's account.

### Book Management

- **Add New Book**
  - `POST /books`
  - Adds a new book to the inventory.

- **List All Books**
  - `GET /books`
  - Retrieves a list of all books.

- **Get Book Details**
  - `GET /books/{bookId}`
  - Retrieves detailed information about a specific book.

- **Update Book Details**
  - `PUT /books/{bookId}`
  - Updates information for a specific book.

- **Delete a Book**
  - `DELETE /books/{bookId}`
  - Removes a book from the inventory.

- **Get Books by Category**
  - `GET /books/category/{category}`
  - Retrieves books filtered by a specific category.

### Order Management

- **Create Order**
  - `POST /orders`
  - Places a new order for books.

- **List All Orders**
  - `GET /orders`
  - Retrieves all orders made by a user or all users (admin).

- **Get Order Details**
  - `GET /orders/{orderId}`
  - Retrieves details of a specific order.

- **Update Order**
  - `PUT /orders/{orderId}`
  - Updates the details of an existing order.

- **Cancel Order**
  - `DELETE /orders/{orderId}`
  - Cancels an existing order.

### Review Management

- **Post a New Review**
  - `POST /reviews`
  - Adds a new review for a book.

- **Get Reviews for a Book**
  - `GET /reviews/{bookId}`
  - Retrieves all reviews for a specific book.

- **Update a Review**
  - `PUT /reviews/{reviewId}`
  - Updates a submitted review.

- **Delete a Review**
  - `DELETE /reviews/{reviewId}`
  - Deletes a review.

### Search Functionality

- **Search Books**
  - `GET /search/books`
  - Searches books by title or author.

- **Search Authors**
  - `GET /search/authors`
  - Finds books by a specific author.

### Reporting and Analytics

- **Sales Data**
  - `GET /analytics/sales`
  - Retrieves sales data analytics.

- **User Engagement**
  - `GET /analytics/user-engagement`
  - Provides metrics on user engagement.

### Admin Endpoints

- **List All Users (Admin)**
  - `GET /admin/users`
  - Retrieves a list of all registered users (admin only).

- **List Out-of-Stock Books**
  - `GET /admin/books/out-of-stock`
  - Lists books that are currently out of stock.

- **Restock Books**
  - `PUT /admin/books/{bookId}/restock`
  - Restocks a specified book.

- **View All Orders (Admin)**
  - `GET /admin/orders`
  - Views all orders placed in the system (admin only).

### Additional Functionalities

- **Submit Feedback**
  - `POST /feedback`
  - Submits feedback regarding the store or service.

- **Get Recommendations**
  - `GET /recommendations`
  - Provides book recommendations based on user's preferences.

- **Subscribe to Notifications**
  - `POST /notifications/subscribe`
  - Subscribes the user to notifications about new books or promotions.
